## Status

These were drafted as part of the Option-C hardening sprint
(2026-04-28) to give SOC 2 readiness a concrete starting point.
They are NOT yet audit-ready — they need:

1. Legal review of language and obligations.
2. Sign-off from each named owner.
3. Evidence collection processes (who attests what, where the
   evidence is filed) for each control.
4. Calibration against an actual SOC 2 readiness assessor's
   findings.
